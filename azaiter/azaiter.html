How to cite source code

My three sources are :-

  • University of Arkansas: Citing Programming Code.
  • University of Bedfordshire: How to cite computer programs and source code.
  • Langara Higher Learning Collage: Citing Programming Code.

In summary: it is very important for a programmer, or a student who's doing a programming course to cite and give credit to the original author whenever he/she (adapt,use,copy) any sort of written code. This could be done by several ways and formats, all of them agree that the title, original author name, date of code creation and type specifications must be included in a proper way within the main code of the person who is citing the code.
